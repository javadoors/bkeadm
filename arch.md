## 3. 依赖关系详解
### 3.1 核心依赖
| 依赖模块 | 用途 | bkeadm 使用场景 |
|----------|------|-----------------|
| `cluster-api-provider-bke` | BKE 核心库 | 集群管理、配置初始化、验证 |
| `k8s.io/api` | Kubernetes API 类型 | CRD 定义、资源操作 |
| `k8s.io/apimachinery` | Kubernetes 对象工具 | 资源序列化、标签选择器 |
| `k8s.io/client-go` | Kubernetes 客户端 | 与 K8s API 交互 |
| `sigs.k8s.io/controller-runtime` | Controller 运行时 | Operator 框架 |
| `sigs.k8s.io/yaml` | YAML 处理 | 配置文件解析 |
### 3.2 容器运行时依赖
| 依赖模块 | 用途 |
|----------|------|
| `github.com/docker/docker` | Docker Engine API |
| `github.com/containerd/containerd/v2` | Containerd 客户端 |
| `github.com/containers/image/v5` | 镜像操作库 |
| `github.com/containers/common` | 容器通用配置 |
